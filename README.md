#  Calculadora de Troco

Este é um pequeno programa em **Python** que calcula o troco de uma compra.  
Ele recebe o valor do produto e o valor pago, e informa se há troco, se o pagamento foi exato ou se o cliente ficou devendo.  
Os valores são destacados em **verde** quando o troco é positivo e em **vermelho** quando há dívida.

---

##  Funcionalidades
- Entrada de valores com vírgula ou ponto.
- Exibição do troco formatado com duas casas decimais.
- Destaque em **verde** para troco positivo.
- Destaque em **vermelho** para valor negativo (dívida).
